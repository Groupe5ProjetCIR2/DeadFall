# DeadFall
GitHub du projet du groupe 5 nommé JuniaOnly. Jeu "DeadFall"

Quelques informations importantes :
- Le projet est divisé en deux dossiers "Lobby" et "Game". A noter que dans Roblox Studio "Game" est une "place" de "Lobby" (autrement dit une sous-expérience)
- Nous avons essayé d'imiter au mieux possible l'arborescence de Roblox Studio.
- ".client.luau" représente un LocalScript dans Roblox Studio
- ".server.luau" représente un Script (server) dans Roblox Studio
- ".luau" représente un ModuleScript dans Roblox Studio
- Nous avons utilisé des dossiers pour représenter divers objets de Roblox Studio (Models, Parts, ou encore des scripts dans des scripts (alors représenté par un fichier init pour le script parent))
